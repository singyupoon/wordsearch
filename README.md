文字搜索遊戲 (Word Search Game)

這是一款專為教學與聖經學習設計的互動遊戲，旨在提供具挑戰性的 30x30 高難度聖經關鍵詞搜索體驗。透過本遊戲，老師或學生可以進行沉浸式的聖經詞彙探索，在尋找隱藏字彙的過程中更深入地默想神的話。

🎮 遊戲連結與特色

立即遊玩： 點擊進入文字搜索遊戲

動態闖關： 自動生成 30x30 的漢字矩陣，並具備自動清理非漢字內容，每次遊玩都是全新的挑戰。

隨機演算法： 支援水平與斜向放置答案，並會隨機打亂目標搜索順序，確保每局遊戲皆有不同的體驗。

沉浸模式： 內建「編輯模式」與「學生模式」，切換時自動隱藏答案，確保遊戲公平性與競賽感。

智慧判定： 演算法會自動檢查詞彙間隔，提升閱讀性；點擊即時標記，提供流暢的遊戲反饋。

列印功能： 內建 PDF 列印輸出，自動優化版面並隱藏遊戲介面，可直接輸出為 A4 練習題。

📋 遊戲方法 (How to Play)

請按照以下步驟進行遊戲：

準備階段 (Setup)：

在左側的「遊戲設定」區域輸入您想要放入遊戲的關鍵詞。

每個詞彙之間用換行或空格分隔即可。系統會自動過濾非中文字符。

開始遊戲 (Start)：

點擊「生成海報」按鈕。系統會隨機排列關鍵詞，並以屬靈詞彙填充空白格。

進入競技 (Student Mode)：

點擊「學生模式」按鈕。系統會自動隱藏答案並打亂目標詞彙順序。

參與者需要按照顯示的「當前搜尋目標」進行查找，並點擊網格進行標記。

驗證與提示 (Verification)：

若遇到困難，點擊「顯示答案」可查看隱藏詞彙位置（切換模式時會自動隱藏，維持公平）。

點擊正確後，詞彙會變為綠色標記，並即時更新遊戲進度 (目標 X / Y)。

輸出成果 (Output)：

點擊「列印為 PDF」按鈕。將遊戲儲存為 PDF，即可獲得一份乾淨的挑戰練習單。

⚙️ 運作機制

碰撞檢測： 採用 isSafe() 函數，確保所有填入的詞彙之間至少保留緩衝距離。

隨機填補： 使用預設的「屬靈文字池」隨機填充空白，讓整張海報保持屬靈氛圍。

響應式矩陣： 基於 CSS Grid 技術，確保在手機、平板或電腦上皆能流暢體驗。
