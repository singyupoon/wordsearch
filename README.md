聖言搜索海報生成器 (Word Search Generator)

旨在快速生成 30x30 高難度聖經關鍵詞搜索海報。透過這個工具，活動籌備人員可以輕鬆生成具有挑戰性且充滿屬靈啟示的文字矩陣。

🛠 功能特色

動態生成： 自動生成 30x30 的漢字矩陣。

自動清理： 輸入內容時會自動過濾所有非漢字（如符號、空格、英數字），確保網格品質。

智能排版： 支援水平與斜向放置答案，大幅提升挑戰難度。

間隔保護： 演算法會自動檢查詞彙間隔，確保所有答案之間至少保留兩個字元的距離，避免文字覆蓋。

雙模式切換： 具備「學生模式」（玩遊戲用）與「關主答案模式」（核對用）。

列印友善： 新增「列印為 PDF」功能，自動優化版面並隱藏設定面板，可直接輸出為 A4 海報。

📋 使用方法

輸入關鍵詞： 在左側的文字區域輸入您想要放入海報的答案，每個詞彙之間用換行或空格分隔即可。

生成海報： 點擊「生成海報」按鈕，系統會根據演算法重新排列詞彙並填充隨機聖經關鍵字。

檢視答案： 點擊「顯示答案」可查看隱藏詞彙的位置，點擊「學生模式」則隱藏標記。

列印輸出： 點擊「列印為 PDF」按鈕，瀏覽器會彈出列印視窗，選擇「儲存為 PDF」即可將海報儲存為實體文件。

⚙️ 運作原理

本工具採用純客戶端 JavaScript 處理，不需要後端伺服器即可運作：

清理機制： 使用正規表達式 /[^\u4e00-\u9fa5]/g 將所有輸入清理為純漢字。

碰撞檢測： 在放置每個詞彙時，會運行 isSafe() 函數，掃描周圍一圈格子。若發現已有詞彙佔用且間隔小於兩個單位，則會自動重新尋找新的座標進行放置。

隨機填補： 網格中未被填入答案的部分，會從預設的「屬靈文字池」（如「神愛世人」、「恩典」等詞彙）中隨機選取字符進行填充，確保海報整體的一致性與屬靈氛圍。

響應式網格： 使用 CSS Grid 與 Tailwind CSS，確保在不同尺寸的螢幕上都能正確顯示 30x30 的矩陣。

PDF 列印技術： 使用 @media print CSS 媒體查詢，強制在列印模式下隱藏控制 UI，並將頁面調整為 A4 直向尺寸，確保 PDF 輸出具備最佳視覺體驗。
