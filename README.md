文字搜索遊戲 (Word Search Game)

這是一個基於 Web 的高難度文字搜索遊戲生成器，專為聖言搜索、語言學習或趣味練習設計。使用者可以輸入自定義的詞彙清單，系統將自動產生 30x30 的字元矩陣。

立即遊玩： 點擊進入文字搜索遊戲(https://singyupoon.github.io/wordsearch/index.html)

主要功能

自動生成矩陣：輸入詞彙後，系統會自動隨機排列，並將剩餘空間以聖經相關詞彙池填滿，確保挑戰性。

雙模式切換：

編輯模式：輸入詞彙、檢視答案，並進行版面設置。

遊戲模式：隱藏答案，開始搜尋任務。點擊正確位置即可標記詞彙。

遊戲進度保護：在遊戲模式下，若點擊「顯示答案」檢視位置，不會直接判定詞彙為已完成；返回遊戲模式後，必須親自點擊正確座標才能推進進度。

全新重置：點擊「重置遊戲」會重新呼叫生成邏輯，生成一張完全不同的文字海報，確保每次挑戰都是新鮮的。

列印功能：針對列印進行最佳化，可直接輸出為練習單。

使用說明

輸入詞彙：在「編輯模式」的文字框中輸入目標詞彙（以空格或換行分隔）。

生成海報：點擊「開始遊戲」即可進入遊戲模式並生成隨機矩陣。

遊玩：根據左側顯示的當前目標詞彙，在右側方格中點擊對應的文字。

輔助功能：

顯示答案：編輯模式中可隨時切換查看目標位置（切換回遊戲模式時會自動隱藏）。

返回編輯：若需修改詞彙或重置，可返回編輯模式。

重置遊戲：將會清空當前進度並重新打亂文字配置，產生全新挑戰。

分享/列印：透過「列印為練習單」按鈕，即可將海報列印或儲存為 PDF。

技術實現

前端框架：原生 JavaScript + Tailwind CSS。

矩陣邏輯：隨機演算法確保詞彙分佈均勻。

響應式設計：自動適應螢幕大小，無論是在電腦或手機上均可順暢遊玩。
